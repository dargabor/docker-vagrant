# Docker-Vagrant

Creates a VM with docker.

## Features
* Using latest docker
* Docker-compose 1.26.0
* Adding code completion to docker and docker compose

## Required
* Virtualbox
* Vagrant

## Vagrant setup
* Used virtualbox provider: `https://app.vagrantup.com/ubuntu/boxes/bionic64`

## Getting started
* Clone git repo
* run `vagrant up`
* run `vagrant ssh`
